Вариант 1. Реализуйте на Haskell функцию classifyNumber, которая классифицирует целое число: \"negative\" для отрицательных, \"zero\" для нуля, \"positive\" для положительных. Вводится список целых чисел.
Вариант 2. Реализуйте на Haskell функцию gradeScore, которая возвращает оценку по баллам: \"A\" для 90+, \"B\" для 80-89, \"C\" для 70-79, \"D\" для 60-69, \"E\" для остальных. Вводится список чисел (от 1 до 100). Программа должна возвращать список строк (String).
Вариант 3. Реализуйте на Haskell функцию isWeekend, которая проверяет, является ли день выходным (суббота=6, воскресенье=7). Функция должна принимать список целых чисел (номеров дней).
